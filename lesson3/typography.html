Typography

Headings

The font for all headings is Cormorant Garamond. This font is a serif font which makes it more elegant but it remains readable for devices. Using a serif font also pyschology alludes to tradition and creates a noticeable contrast for the paragraph font.

Body Text

For the body text Lato is the font that is being used. This font is a sans serif font. A sans serif font provides a positive UX design for a website as it's easy on the eyes esecially in paragraphs. This font is also from a different font family than the heading font which creates a visual appealing contrast.
